### FTC Gradle Setup

Add JitPack to your repositories:

```gradle
repositories {
    maven { url 'https://jitpack.io' }
}
```
Then add the following to your build.gradle:

```
dependencies {
    implementation 'com.github.Loonybot:Sidekick:lib-v1.2.3'
}
```
